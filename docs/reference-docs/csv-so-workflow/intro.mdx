# 工作流简介

该工作流是通过读取Excel表格来导出每个Sheet的数据，然后以单个Sheet为基础导入到Unity中生成类，最后以数据类的形式来生成包含所需数据的SO。  
特点：
- Excel会转换成csv，既能保证数据不丢失，又能使每个Sheet的数据轻量化
- Unity友好，可以在编辑器中直接看到csv格式的文件
- 支持多格式，包括`int`，`float`，`bool`，`enum` 等

## 日后的更新与扩展
